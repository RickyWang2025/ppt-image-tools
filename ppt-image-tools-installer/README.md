# PPT图片处理工具 - 安装包

## 一键安装

**双击 `一键安装.bat` 即可完成所有安装！**

安装程序会自动：
1. 检测Python环境
2. 安装程序文件到 `%LOCALAPPDATA%\PPTImageTools`
3. 安装Python依赖包
4. 注册PPT插件
5. 创建桌面快捷方式

---

## 使用方法

### 第一步：启动服务
双击桌面上的 **"PPT图片工具"** 快捷方式

### 第二步：打开PPT插件
1. 打开 PowerPoint
2. 点击 **插入** → **获取加载项**
3. 点击 **管理我的加载项**
4. 选择 **"图片处理工具"**

### 第三步：处理图片
1. 在PPT中选中要处理的图片
2. 在插件面板中选择功能：
   - **放大**：高清放大2-4倍
   - **抠图**：智能移除背景
   - **擦除**：移除水印/物体
   - **矢量**：图片转SVG

## 常见问题

**Q: 提示"未检测到Python"？**
A: 请先安装Python：https://www.python.org/downloads/
安装时务必勾选 "Add Python to PATH"

**Q: 首次使用很慢？**
A: 首次运行需要下载AI模型（约500MB），请耐心等待

**Q: 插件列表中没有"图片处理工具"？**
A: 安装后请重启PowerPoint，然后检查：
文件 → 选项 → 信任中心 → 信任中心设置 → 受信任的加载项目录
